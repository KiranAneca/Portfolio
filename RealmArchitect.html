Realm Architect

Overview

This application is a tool for dungeon masters to use for both the setup and the running of campaigns in a D&D setting, made by a group of six people. It consists out of different parts that work together to provide all the necessary tools needed to let your creativity flow wild and help you keep track of the game you are running.

Codex

The codex is where all the info is located. The DM can quickly find all the info he needs. Everything can be quickly found using a combination of filters, tags and a searchbar.

Token Menu

The token menu, display the healthbar, AC and the name of the token. Here the health can be changed, and status effects can be applied. Usefull for keeping track of combat.

Effects Creator

Here the user can create effect to apply on the map and change the scale of these visual effects.

Markers

The marker tool allows the DM to add notes to the map on the fly, ready to be used in the next session. There are a variety of markers, to quickly distinguish between them. These also have the funcionality to lead into other maps, to quickly link different maps together.
