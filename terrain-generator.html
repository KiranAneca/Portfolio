Terrain generator

A tool for quickly generating maps.

This is a tool that was designed to quickly generate world maps, with both video games and tabletop games like D&D in mind.

This version is a prototpye, but I am planning on making a fully fledged tool of this project, with more features.

Art

The artstyle is very low poly and stylized.

Technical

This project is written in C# using the unity engine.

The art assets were made in 3ds max.

See in github

Settings

The parameters influence the way the map is generated and offers the user to change how the map looks.

You can change:

- The size of the map
- The biome sizes
- The size of the ocean, beaches, plains and mountainranges
