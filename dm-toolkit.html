DM Toolkit

A program for DM's to manage their campaigns.

This project provides all the tools a DM could want to both set up his game, and to run it.

Codex

The codex is where all the info is located. The DM can quickly find all the info he needs. Everything can be quickly found using a combination of filters, tags and a searchbar.

Character creator

This tool allows the quick creation of characters for your player.

Markers

The marker tool allows the DM to add notes to the map on the fly, ready to be used in the next session. There are a variety of markers, to quickly distinguish between them.

Technical

This project is written in C# using the unity engine.

This project is atively being developed by a group of six people.
