TrickShot

A puzzle game where precision is key.

This is a small game I am currently developing.
It is a puzzle game, where every level is a pool table with a certain layout off balls.
Your job is to use a variety of tricks to pot all the balls in the least amount of strokes.

The tricks

These are the types of tricks you come equiped with to try and clear every level.

Phase shot

This shot allows the ball to phase trough the first ball it hits, ignoring collision with that ball.

Reverse shot

The first time this shot hits another ball, the balls get swapped places and velocities.

Technical

This project was written in C# using the unity engine.
